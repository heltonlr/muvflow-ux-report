# MuvFlow — Relatório de Análise UX

Análise completa de experiência de uso (UX) da plataforma MuvFlow, cobrindo landing page, fluxo de criação de conta e fluxo de criação de música.

**Data da análise:** 30/03/2026

---

## Como Usar

Abra o **`Relatorio_UX_MuvFlow.html`** no navegador. É o relatório consolidado e interativo com navegação visual, scores e todos os achados.

O relatório cobre três fluxos:

| Fluxo | O que cobre |
|-------|-------------|
| Landing Page | LP + Explorar, Indique e Ganhe, Criar Música, Calculadora de Royalties |
| Criar Conta | Signup e pós-signup |
| Criar Música | Hub de seleção + 4 formulários de criação |

## Principais Achados

- Números inconsistentes entre páginas minam credibilidade
- Dark patterns de escassez artificial em produto gratuito
- Omissão sobre cessão de direitos autorais
- Navegação fragmentada e incoerente
- Foco desproporcional em monetização (~80% do conteúdo)
